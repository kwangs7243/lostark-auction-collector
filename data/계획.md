
# Lost Ark 상급 아비도스 제작 최적화 프로젝트 진행 정리

## 프로젝트 목표

로스트아크 거래소 API 가격 정보를 기반으로
상급 아비도스 융화재료 제작 시 가장 효율적인 제작 플랜을 계산한다.

핵심 목표:

1. 보유 재료만으로 제작 가능한지 확인
2. 보유 재료 교환만으로 제작 가능한지 확인
3. 교환 후 부족분 구매 시 최소 구매 비용 플랜 계산
4. 최종적으로 가장 효율적인 플랜 선택

# 앞으로 남은 가장 큰 구현

## 구매 + 교환 최적화 플랜

현재 가장 큰 산.

아직 미구현.

목표:

```text
교환할 재료조차 부족할 경우
무엇을 구매해서 교환하는 것이 가장 싼가?
```

예시 상황:

```text
부드러운 목재 직접 구매

vs

목재 구매
→ 가루 교환
→ 부드러운 목재 교환
```

또는:

```text
목재 구매
→ 가루
→ 아비도스 목재
```

현재 필요한 핵심:

```text
"직접 구매가 싼가?"
vs
"교환 경유가 싼가?"
```

비교 시스템.

# 최종 목표 플랜 흐름

## 1단계

```text
보유 재료만으로 제작 가능한가?
```

가능:

```text
즉시 종료
```

---

## 2단계

```text
보유 재료 교환만으로 제작 가능한가?
```

가능:

```text
사용 재료 가치가 가장 적은 플랜 선택
```

---

## 3단계

```text
교환 후 부족분 구매
```

목표:

```text
최종 구매 비용 최소화
```

현재 구현 거의 완료.

---

## 4단계 

```text
구매시 교환까지 고려한 최적화
```

목표:

```text
무엇을 구매해서
어떤 교환 루트를 타야
최종 비용이 가장 싼가?
```

예상 필요 요소:

* 구매 후 교환 후보 생성
* 교환 경로 비교
* 가격 우선순위 비교
* 다단계 교환 비교

# 현재 프로젝트 상태 평가

현재는 단순 계산기 수준을 넘어:

```text
가격 기반 전략 생성
+
후보 생성
+
최적 플랜 선택
```

구조까지 진입한 상태.

핵심 구현 난관은:

```text
구매 + 교환 경로 최적화
```

쪽으로 이동 중.

전체 흐름은 이미 상당히 완성 단계에 가까워지고 있음.
