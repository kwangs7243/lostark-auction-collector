# 상급 아비도스 제작 계산기 설계

## 1. 목표

보유 재료와 실시간 시세를 기준으로
지정한 제작 횟수만큼 제작 가능한 최저 비용 계획을 계산한다.

기본 제작 횟수는 40회이지만,
모든 계산 함수는 craft_count 값을 받아
임의의 제작 횟수에서도 동작해야 한다.

# 3. 제작 레시피

## 1회 제작 필요 재료

- 부드러운 목재: 59

# 4. 교환 레시피

## 재료 → 생활의 가루

- 목재 100개 → 생활의 가루 80개
- 부드러운 목재 50개 → 생활의 가루 80개

## 생활의 가루 → 재료

- 생활의 가루 100개 → 아비도스 목재 10개
- 생활의 가루 80개 → 부드러운 목재 50개

# 6. 계획 종류

## 6-1. 직접 구매 플랜

부족한 재료를 그대로 구매한다.

예:

- 부족한 목재 → 목재 구매
- 부족한 부드러운 목재 → 부드러운 목재 구매
- 부족한 아비도스 목재 → 아비도스 목재 구매

### 필요 함수

```python
calculate_direct_purchase_plan()
```

---

## 6-2. 보유 재료 교환 플랜

추가 구매 없이,
남는 재료만 교환하여 제작을 시도한다.

예:

- 남는 목재 → 생활의 가루
- 생활의 가루 → 아비도스 목재

### 필요 함수

```python
calculate_required_powder_for_missing_abidos()
calculate_exchange_plan_for_powder()
calculate_owned_exchange_plan()
```

---

## 6-3. 구매 후 교환 플랜

재료를 직접 구매하지 않고,
더 저렴한 재료를 구매 후 교환한다.

예:

- 아비도스 목재 직접 구매
vs
목재 구매 → 가루 → 아비도스 목재

---

## 6-4. 부드러운 목재 교환 플랜

목재가 많이 남을 경우,
부드러운 목재를 직접 구매하지 않고
생활의 가루 교환으로 해결한다.

예:

- 목재 → 생활의 가루
- 생활의 가루 → 부드러운 목재

---

## 6-5. 혼합 플랜

여러 교환과 구매를 조합한다.

예:

- 일부는 직접 구매
- 일부는 교환
- 일부는 보유 재료 사용

---

## 6-6. 최저 비용 플랜

모든 계획을 계산 후,
총 비용이 가장 낮은 계획을 선택한다.
